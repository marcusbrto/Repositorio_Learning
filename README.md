## Repositório de Projetos – Curso de Algoritmos

---

### Descrição

Este repositório reúne **projetos desenvolvidos durante o curso de Algoritmos**, com foco em **aprendizado**, **prática** e **evolução contínua** em programação e lógica computacional.

Os exercícios e projetos presentes aqui representam minha progressão ao longo do curso, servindo tanto como material de estudo quanto como registro da minha evolução na área de tecnologia.

---

### Ferramenta utilizada

* **Visualg 3.0.7**

Todos os algoritmos foram desenvolvidos e testados utilizando essa ferramenta.

---

### Objetivos do repositório

* Praticar lógica de programação
* Consolidar conceitos fundamentais de algoritmos
* Criar um histórico de aprendizado
* Facilitar revisões e melhorias futuras

---

### Acesso ao repositório

O projeto está disponível no GitHub:

* [https://github.com/marcusbrto/Algoritmos](https://github.com/marcusbrto/Algoritmos)

---

### Como clonar o repositório

Para baixar o repositório na sua máquina local, utilize o comando:

```bash
git clone https://github.com/marcusbrto/Algoritmos.git
```

Após isso, basta acessar a pasta clonada e abrir os arquivos no **Visualg**.

### Observação

Este repositório tem caráter **educacional** e está em constante atualização conforme novos conteúdos e exercícios são estudados.
